# stash-plugins

A personal collection of [Stash](https://github.com/stashapp/stash) plugins.

## Installation

1. In Stash go to **Settings → Plugins → Available Plugins → Add Source**
2. Name: `g4clips plugins`
3. URL: `https://g4clips.github.io/stash-plugins/index.yml`
4. Browse and install plugins from the list

## Plugins

| Plugin | Description |
|--------|-------------|
| [Data18StashDB](plugins/Data18StashDB/) | Scrape Data18 and match scenes against StashDB |
